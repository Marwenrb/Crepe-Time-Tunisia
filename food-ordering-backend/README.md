# Crêpe Time Tunisia — Backend API

API REST pour la gestion des commandes, restaurants, et utilisateurs.

## Stack

- Node.js + Express
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Cloudinary (images)
- WhatsApp Notifications

## Démarrage

```bash
cp .env.example .env
npm install
npm run dev
```

Voir le [README principal](../README.md) et le [Guide Développeur](../DEVELOPER_GUIDE.md) pour la documentation complète.
