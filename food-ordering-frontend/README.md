# Crêpe Time Tunisia — Frontend

Application React pour la commande de crêpes artisanales en ligne.

## Stack

- React 18 + TypeScript
- Vite 7
- Tailwind CSS + Shadcn/ui
- React Query (TanStack)
- React Router DOM

## Démarrage

```bash
cp .env.example .env
npm install
npm run dev
```

Voir le [README principal](../README.md) et le [Guide Développeur](../DEVELOPER_GUIDE.md) pour la documentation complète.
